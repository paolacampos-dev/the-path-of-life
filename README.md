## ğŸ”® How It Works

1. The user is greeted and asked for their name.
2. The user selects a number between **0 and 22**.
3. Each number maps directly to a Major Arcana card stored in a JavaScript array.
4. The selected card is revealed with a symbolic message.
5. The page visually presents all Major Arcana cards in order, styled as individual sections.
